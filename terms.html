1. 목적

본 약관은 HealthArchive가 제공하는 공개 자료, 검색·비교 도구 및 승인 사용자용 보호 자료의 이용 기준을 정합니다.

2. 정보의 성격

사이트의 자료와 분석 결과는 실무 검토를 돕기 위한 참고 정보이며 법률, 규제, 의료 또는 투자 자문이 아닙니다. 자동 수집·분류된 자료에는 누락, 지연 또는 오류가 있을 수 있습니다.

3. 이용자의 확인 의무

신청, 신고, 제품 개발, 계약 또는 안전성 판단에 사용하기 전 최신 법령·고시·공전·기관 원문과 원본 논문을 직접 확인해야 합니다. 이용자의 최종 판단과 업무 적용에 대한 책임은 이용자에게 있습니다.

4. 보호 자료 접근

보호 자료는 운영자가 승인한 이메일로 인증한 이용자만 열람할 수 있습니다. 인증번호, 로그인 링크 또는 접근 권한을 타인과 공유하거나 권한 통제를 우회해서는 안 됩니다. 필요 시 접근 권한을 제한하거나 철회할 수 있습니다.

5. 금지행위

서비스 장애를 유발하는 자동 요청, 보안 우회, 보호 자료의 무단 대량 수집·재배포, 타인의 권리 침해, 불법 목적의 이용을 금지합니다.

6. 외부 자료와 링크

출처가 표시된 외부 자료의 권리는 각 제공자에게 있습니다. 외부 사이트의 내용과 가용성은 해당 운영자의 정책을 따릅니다.

7. 서비스 변경 및 중단

자료 갱신, 보안 조치, 외부 API 변경 또는 운영상 필요에 따라 기능을 수정하거나 일시 중단할 수 있습니다. 중요한 변경은 사이트에 공지합니다.

8. 책임 범위

고의 또는 중대한 과실이 없는 한, 참고 정보의 이용이나 외부 서비스 장애로 발생한 간접 손해에 대해 책임을 부담하지 않습니다. 관련 법령상 배제할 수 없는 책임은 제외됩니다.
